################################################################
## 一、用户相关接口
################################################################

#### 1. 获取验证码

POST	/auth_codes/fetch

##### 请求参数

mobile	手机号, 必须

##### 响应结果
{ code: 0, message: 'ok' }

------------------------------------------------------------------------------

#### 2. 用户登陆

POST	/account/login

##### 请求参数

mobile	手机号, 必须
code    手机验证码, 必须

##### 响应结果
{ code: 0, message: 'ok', data: 用户个人信息 }

------------------------------------------------------------------------------

#### 3. 获取用户个人资料

GET		/user/me

##### 请求参数

token	认证token, 必须

##### 调用示例

GET		/user/me?token=xxxxxxxxx

##### 响应结果
{ code: 0, message: 'ok', data: 用户个人信息 }

------------------------------------------------------------------------------
